# Factory2 Research Scripts

Historical Factory2 proof and gap-search scripts should eventually move under this directory.

They remain at top-level `scripts/` for now because the current test suite imports those module paths directly. Treat the product validation entry points as:

- `scripts/validate_video.py`
- `scripts/register_test_case.py`
- `scripts/start_factory2_demo_stack.py`
- `scripts/capture_factory2_app_run_events.py`
- `scripts/build_human_truth_ledger_from_csv.py`
- `scripts/compare_factory2_app_run_to_truth_ledger.py`
- `scripts/compare_app_run_to_human_total.py`
- `scripts/preview_video_frames.py`
- `scripts/select_video_frames.py`
