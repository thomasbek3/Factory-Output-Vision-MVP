# Operator Runbook

This runbook is for the local dashboard experience. It avoids validation internals.

## Start Monitoring

1. Open the dashboard URL.
2. Confirm the source shown on the page is the expected camera or demo video.
3. Click `Start monitoring`.
4. Confirm Runtime Total starts at `0`.
5. Watch for count increments when finished products are placed in the output/resting area.

## Normal Status

- Green means the line is running at or above the expected rate.
- Yellow means the line may be slowing down.
- Red means the line appears stopped or below threshold.

## Corrections

Use manual correction controls only as operator oversight. Corrections are not proof that automated validation passed.

## Support

Use the troubleshooting page and support bundle when the source disconnects, the count is visibly wrong, or the dashboard is stale.
